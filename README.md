# CampusFlow

## 📌 Sobre o projeto

Projeto web desenvolvido para otimizar a organização acadêmica de estudantes, permitindo o gerenciamento centralizado de tarefas, disciplinas, calendário de entregas e rotinas de estudo através de uma interface intuitiva e dinâmica.

## 🚀 Funcionalidades

* Dashboard interativo com resumo diário e alertas de próximas entregas[cite: 1]
* Cadastro e gerenciamento de Tarefas com definição de prioridades (Alta, Média, Baixa) e prazos[cite: 1]
* Organização de Disciplinas com cálculo automático de progresso baseado nas tarefas concluídas[cite: 1]
* Calendário mensal dinâmico (estilo Google Calendar) integrado automaticamente com os prazos cadastrados[cite: 1]
* Modo Foco integrado com Timer Pomodoro e checklist de etapas para tarefas urgentes[cite: 1]
* Aba de Insights com estatísticas de desempenho e um Chat simulado com IA para dicas de organização[cite: 1]
* Busca global rápida e personalização de perfil (nome, curso e paleta de cores do sistema)[cite: 1]

## 🧠 Conceitos e Tecnologias utilizados

* **HTML5:** Estruturação semântica do sistema[cite: 1]
* **CSS3:** Estilização responsiva utilizando CSS Variables, CSS Grid Layout, Flexbox e animações nativas (`@keyframes`)[cite: 1]
* **JavaScript Vanilla:** Manipulação direta de DOM, gerenciamento de estado local utilizando Arrays e Objetos, métodos de iteração (`map`, `filter`, `reduce`, `sort`) e manipulação de datas[cite: 1]
* **Arquitetura SPA (Single Page Application):** Navegação fluida entre telas simulada via JavaScript sem necessidade de recarregar a página ou utilizar frameworks externos[cite: 1]
